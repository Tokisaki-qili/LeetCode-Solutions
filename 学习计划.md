## 🚀 第一个Python项目：分析你的LeetCode刷题数据

当你学完NumPy/Pandas后，可以做这个项目练手：

```python
import pandas as pd
import matplotlib.pyplot as plt

# 你的刷题记录
data = {
    "日期": ["7月第1周", "7月第2周", "7月第3周", "7月第4周"],
    "简单题": [3, 5, 4, 6],
    "中等题": [0, 1, 2, 3]
}

df = pd.DataFrame(data)
df["总计"] = df["简单题"] + df["中等题"]

# 画图
plt.plot(df["日期"], df["总计"], marker="o")
plt.title("LeetCode刷题趋势")
plt.xlabel("时间")
plt.ylabel("题数")
plt.show()
```
