## 💡 刷题技巧

1. **独立思考15分钟** → 想不出来就看题解
2. **理解思路后关掉题解** → 自己重写一遍
3. **写注释** → 把思路写在代码里，方便复习
4. **上传GitHub** → 每天提交，积累绿点
